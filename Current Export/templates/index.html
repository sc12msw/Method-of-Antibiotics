How to get the Data in XML

For use in your application or just to view the data from the database in XML just use the URL http://localhost8080/cfmedicine/rest/cfmed. This will return the database in XML format. The output can be seen by clicking the button below

To Access the Administration Interface Adding Drugs or Pathogens.

To access the administration interface to add a drug you need to have the administrators log in and password. Once this has been given you can add drugs using the form provided under the add drugs tab. Or create your own form to send a post request to the server to the URL "http://localhost8080/cfmedicine/rest/secure/addDrug". The fields required are:

  • name
  • type
  • brand This can be used do add up to 5 brand names starting from 0 "brand" to "brand4".
  • sideEffects
  • indication
  • interactionName interactionDescription This can be used do add up to 10 interactions starting from 0 "interactionName" and "interactionDescription" to "interactionName9" and "interactionDescription9".
  • The routes of administration are oral,iV,inhaled,iM,pR,sC. Usage (route)(Adult/Paediatric)(Admin/Dose) Examples: oralAdultAdmin iVPaediatricDose

To access the administration interface to add a pathogen you need to have the administrators log in and password. Once this has been given you can add pathogens using the form provided under the add pathogen tab. Or create your own form to send a post request to the server to the URL "http://localhost8080/cfmedicine/rest/secure/addPathogen". The fields required are:

  • name
  • fLDrug This can be used do add up to 10 first line drugs starting from 0 "fLDrug" to "fLDrug9".
  • description

Administration Interface View/Delete

To view the drugs or pathogens use the tabs again a log in is needed as an administrator. To delete a drug or pathogen use the interface or you can send a Delete request with the ID of the drug or pathogen to the following URLs:

  • Drug - http://localhost8080/cfmedicine/rest/secure/drugs/{id}
  • Pathogen - http://localhost8080/cfmedicine/rest/secure/{id}
